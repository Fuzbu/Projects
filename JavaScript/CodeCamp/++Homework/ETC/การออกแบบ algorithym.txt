

1. Array - เป็นการเก็บข้อมูลแบบ fix คือ สร้างแล้วสร้างเลย ไม่สามารถเพิ่มหรือลดได้
   Link List - เก็บข้อมูลแบบเป็นลำดับขั้นก่อน-หลัง และสามารถเพิ่มหรือลดข้อมูลได้เรื่อยๆ


2.  ข้อมูลอยู่ใน Node
ตัวต้นไม้หลักเรียกว่า Root Node และมีกิ่งก้านเป็น Link
ในปลายกิ่งก้าน คือ node ข้อมูลอื่นๆ ถูกแบ่งเป็น n กลุ่มต่างๆ เรียกว่า sub tree 

เหมือนกับต้นไม้สามารถตัดกิ่งหรือเพิ่มกิ่งก้านได้อยู่เรื่อยๆ จึงนำไปปฎิบัติงานได้ง่าย และเห็นโครงสร้างที่ชัดเจน


3. การที่ Front และ Rear ชี้ไปที่ตำแหน่งเดียวกัน แปลว่า คิวนั้นเต็มแล้ว 
ทราบได้เพราะว่า Rear คือจุดตำแหน่งสูงสุดของคิวนั้นๆ ไม่สามารถไปต่อได้ มันจึงชี้ไปที่เดียวกัน
