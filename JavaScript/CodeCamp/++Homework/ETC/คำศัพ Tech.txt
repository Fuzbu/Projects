สุทธิธรรม จำปาทอง


1.Pixel - จุด ที่เป็นจุุดภาพที่แสดงบนหน้าจอ และเอามารวมกันให้เกิดเป็นภาพขึั้นมา 

2.Pop-up blocker - โปรแกรมป้องกัน Pop-up ต่างๆตามเวปไซด์ และโปรแกรมต่างๆ เช่น สื่อโฆษณา

3.Protocol - ระบบในการทำงาน

4.Proxy - server ที่ผู้ใช้ต้องผ่านก่อนที่จะเข้าไปหาข้อมูลใน Internet

5.Sever - เป็นคอมพิวเต้อแหล่งเก็บข้อมูล และ ให้ข้อมูล กับเครื่องคอมพิวเต้อตัวอื่นๆ

6.Software - เป็นคำเรียกของโปรแกรมที่ถูกสร้างขึ้นโดยคอมพิวเต้อ

7.TrueType - เป็นฟอนต์ประเภทหนึ่ง ที่มีมาตรฐานของฟอนต์ ซึ่งถูกวางรากฐานมาจาก Apple และ Microsoft

8.UNIX - เป็นระบบปฎิบัติทางคอมพิวเต้อการรุ่นแรกๆของโลก

9.Virtualization - เป็นเครื่องมือที่ใช้การรันระบบปฎิบัติการหลายๆตัวในเครื่องคอมพิวเต้อตัวเดียว เช่น Windows กับ IOS

10.VPN - เป็นเหมือนท่อ Internet ส่วนตัว หลักๆเอาไว้ใช้เข้า เวปไซต์ต่างประเทศ ที่ถูกปล๊อค IP

11.WLAN - เป็นเครื่องมือที่ใช้ส่ง รับ ข้อมูลแบบไร้สาย เช่น WIFI

12.Zip - เป็นซอฟแวร์ที่เอาไว้รวบรวมข้อมูลไว้ในที่เดียว ซึ่งสามารถบีบอัดข้อมูลในเล็กกว่าขนาดปกติ
