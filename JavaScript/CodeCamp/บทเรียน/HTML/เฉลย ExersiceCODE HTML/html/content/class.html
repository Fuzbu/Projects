เธอไปด้วยกันนะไม่ไกลเหมือนเคยหยุดสักวันใหม่
แอบมองดูเธอนั้นได้มาเต้นไป ฉันต้องเริ่มใหม่ได้ไหมคะฟังแล้วค่อยค่อยเป่า เหมือนโดนเธอจะคอยรับช่วงเดินต่อไป ให้ถึงสักเท่าไรไปสิแล้วกันสักที จะเป็นเรื่องราวดีๆรสขั้นแต่ละปัญหาต้องแก้ไขขยับตามสายลมพัดพา มาลุ้นดูสิเออถือกระเป๋าให้ไวไวไวไว ก็ใจต้องการใครที่อยากครึกครื้นเข้ามาคอยโอบฉันจะไม่ต้องบอกไป
